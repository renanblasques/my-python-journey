# Substituição de palavras na string

Faça um programa que leia uma string e duas palavras (pal1 e pal2). O programa deve criar e imprimir outra string onde cada ocorrência da pal1 é substituída pela pal2.

## Exemplo

### Entrada

```
Casa Portao Predio Casa Piso Casa Telhado
Casa
Apto
```

### Saída

```
Apto Portao Predio Apto Piso Apto Telhado
```
