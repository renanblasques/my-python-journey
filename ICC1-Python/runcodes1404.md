# Listando os números primos

Criar um programa que lê um número e gerar todos os números primos entre 2 e o número fornecido pelo usuário, escrevendo na tela o resultado.

Obs: criar uma função que verifica se um número é primo para uso no programa. Se o número for primo, a função retorna 1. Caso contrário, retorna 0.

## Exemplo

### Entrada

```
20
```

### Saída

```
2
3
5
7
11
13
17
19
```
